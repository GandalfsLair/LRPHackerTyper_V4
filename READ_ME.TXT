This code uses iFrames to embed other HTML into the same screen.

In this instance the iFrames point to the various hackertypers.


How to set up
Have the file saved in a location.
Add the various HackerTypers within folders.
Ensure that the hackertyper.html file is referenced correctly in the main file (be sure to include the folder/file.html!)
Amend the name of the title after the iFrame tage so that the window is labelled.

How to Use
Clicking into the iFrame window activates that window.
Key presses will now be taken from the respective text file and displayed to screen.
Hitting 1 - 0 three times will trigger the pop up in the active screen only.
Escape will clear the pop up but only for the active window.
